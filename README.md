# Manga Text Detector

A CNN-transformer based text detector for manga pages. Trained on one-punchman manga pages.
The model predicts character heatmaps and affinity heatmaps. The affinity heatmaps correspond to finding what characters have an affinity for eachother and the character heatmaps finds those characters.
